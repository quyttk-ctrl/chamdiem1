✏️ Chấm Từng Sinh Viên

Điền thông tin sinh viên, chọn mã đề, và dán đáp án để tính điểm.

📖 Hướng Dẫn Sử Dụng

1️⃣ Bước 1: Cấu Hình Đáp Án

• Vào tab "⚙️ Cấu Hình Đáp Án"
• Bấm "➕ Thêm Mã Đề Mới"
• Nhập tên mã đề và 55 đáp án (A, B, C, D)
• Bấm "💾 Lưu Tất Cả"

2️⃣ Bước 2a: Chấm Hàng Loạt (Khuyên Dùng)

• Vào tab "📥 Chấm Hàng Loạt"
• Upload file Excel chứa danh sách SV
• File Excel cần có các cột:
    - Mã SV (hoặc StudentID, ID...)
    - Họ Tên (hoặc Name, Tên...)
    - Mã Đề (hoặc Code, ExamCode...)
    - Đáp Án (hoặc Answers, Answer...)
• Chọn cột "Mã Đề" và "Đáp Án"
• Bấm "🧮 Chấm Tất Cả" → kết quả hiển thị ngay
• Bấm "📥 Xuất File Excel" để tải kết quả

2️⃣ Bước 2b: Chấm Từng Sinh Viên

• Vào tab "✏️ Chấm Từng Sinh Viên"
• Điền: Mã SV, Họ tên, chọn mã đề
• Dán 55 đáp án
• Bấm "🧮 Tính Điểm"
• Bấm "📥 Xuất File Excel"

💡 Mẹo Sử Dụng

✓ Dữ liệu cấu hình lưu trong trình duyệt
✓ File Excel cột "Đáp Án" có thể là: "A B C D..." hoặc "ABCD..." (tự động xử lý)
✓ Tên cột không phân biệt hoa/thường
✓ Đáp án chỉ nhận A, B, C, D (không phân biệt hoa/thường)
✓ Xuất kết quả để lưu trữ hoặc chia sẻ
